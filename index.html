DeadLogs is a tool/template for making static blogs and sites. It's similar to and inspired by Zonelets.

Since DeadLogs relies solely on plain HTML, CSS, and vanilla JavaScript, making it static, it's able to be viewed locally without having to install extra technologies and can be hosted freely on plenty of hosting services.

You can check out this website's source code to see how a DeadLogs site can be set up.
