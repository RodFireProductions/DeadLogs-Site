Download DeadJournals

First thing first is downloading the deadjournals_(theme).zip file. This can be downloaded on the itch.io page or right here. The current theme options: . Note: If you plan on doing all of your own css, it's recommended you start with an Absent theme.

Once the zip is downloaded, you need to unzip it to a desired location.

-❁-

Set Up Basic Configurations

Once you've got everything unzipped, you need to begin setting up your script.js file. As you can probably see, that file is riddled with comments. Comments are everywhere. Those comments will explain what each variable does.

I'd suggest filling out the first 36 lines of that file to go ahead and get some of the basic page generation features of DeadJournals customized to you.

Do you really have to know html to use DeadJournals?
I mean you don't technically have to; you could use a wysiwyg (what you see is what you get) editor that converts to html or use a markdown to html converter if you can't or don't want to write the html code yourself.
